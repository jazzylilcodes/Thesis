## Tabel headings for protein level abundances
- RUN: LC-MS/MS run
- Protein: uniprot accession number
- Logintensities: MS stats summarised log2 intensity
- origionalRUN: origional run name
- GROUP: experimental condition
- SUBJECT: sample donor ID
- TotalGroupMeasurements: total number of quantified peptide measurements to calculate protein abundance
- NumMeasuredFeature: Number of features measured
- MissingPercentage: Number of missing features as a perfentage
- more50missing: Missing values TRUE if over 50% of features missing
- NumImputedFeature: Number of features imputed
